# To-Do List Application (Python)

A simple console-based To-Do List app built using Python.  
You can add tasks, view tasks, and remove tasks.  
All tasks are saved in a text file so they stay even after closing the program.

## Features
- Add Task
- View Tasks
- Remove Task
- Saves tasks in tasks.txt automatically
- Easy menu-based interface

## How to Run
1. Install Python
2. Open terminal in the project folder
3. Run:

python todo.py

## Files in this Project
- todo.py
- tasks.txt (auto-created)
- README.md

## Description
This project uses:
- Lists for storing tasks
- File handling with open()
- strip() to clean input
- pop() to remove tasks
- with-statement for safe file handling
